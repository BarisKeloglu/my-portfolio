# Plantify 🌿

Yapay zeka destekli bitki sağlığı izleme ve akıllı sulama hatırlatıcıları içeren sanal bahçe asistanı mobil uygulaması.

## 🚀 Projeyi Bilgisayarınızda Test Etme Adımları

Projeyi yerelde sorunsuz çalıştırmak ve test etmek için aşağıdaki adımları takip edebilirsiniz:

1. **Projeyi İndirin:** Sağ üstteki yeşil `Code` butonuna tıklayıp `Download ZIP` diyerek tüm portfolyoyu bilgisayarınıza indirin ve klasöre çıkartın.
2. **Android Studio ile Açın:** Android Studio'yu açın, `File -> Open` adımlarını takip ederek `my-portfolio` klasörü altındaki `Plantify` klasörünü seçin.
3. **Gradle Senkronizasyonu:** Proje açıldığında Android Studio'nun projenin çalışması için gerekli Gradle bağımlılıklarını indirmesini bekleyin (Sağ alttaki yükleme barının bitmesini bekleyin).
4. **Yapay Zeka Modülü:** Uygulamanın yapay zeka destekli bitki sağlığı izleme modülünün yerelde sorunsuz analiz yapabilmesi için emülatörünüzün veya test cihazınızın internet bağlantısının aktif olduğundan emin olun.
5. **Çalıştırma:** Üst menüdeki yeşil **Run (Oynat)** butonuna basarak uygulamayı sanal veya gerçek bir Android cihazda hemen test edebilirsiniz.

## 🛠️ Kullanılan Teknolojiler
* **Dil:** Kotlin
* **Mimari:** MVVM / Clean Architecture
* **Öne Çıkan Özellikler:** AI Tabanlı Bitki Sağlığı Analizi, Akıllı Bildirimler (Sulama/Bakım Zamanı)
